# NEUROTRACK - AI BASED DIGITAL BEHAVIOR ANALYZER

This project is developed as a Data Science project to analyze personal digital behavior using machine learning techniques. In this project, a custom synthetic dataset was created which includes features such as screen time, sleep duration, stress level, physical activity, caffeine intake, and phone usage before sleep.

We performed data preprocessing and exploratory data analysis to understand the relationship between lifestyle factors and stress. After that, we introduced a custom Digital Risk Score and Lifestyle Score to measure user behavior patterns.

A machine learning model (Random Forest Regressor) was used to predict stress levels based on user activity. We also applied anomaly detection using Isolation Forest to identify unusual behavior patterns.

Additionally, a simple recommendation system was developed to provide suggestions like reducing screen time, improving sleep, and increasing physical activity.

The results show that higher screen time and lower sleep are strongly associated with increased stress levels, while physical activity helps reduce stress.
